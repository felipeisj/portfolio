# Felipe Salazar — Portfolio

Portfolio personal: experiencia, proyectos y contacto.

Construido con:

- **Next.js** (App Router, React 19)
- **TypeScript**
- **Tailwind CSS v4**
- **GSAP** (animaciones de scroll)

## Estructura

- `/src/app`: layout raíz, página principal, favicon y OG image generados
- `/src/components`: secciones de la página (Nav, Hero, Experience, Projects, About, Contact, Footer)
- `/src/lib/data.ts`: contenido editable (experiencia, proyectos, stack)

## Desarrollo

```bash
npm install
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
